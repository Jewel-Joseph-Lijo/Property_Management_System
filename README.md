This is a Property Management System developed using Django, HTML5, CSS3, Bootstrap 5, JavaScript, and SQLite3. This platform simplifies core rental operations with features like property listings, tenant onboarding, rental applications, maintenance request tracking, rent due date management, and auto-generated payment receipts (PDF). With a clean UI and role-based access for Admin and Tenant, the system delivers a smooth and efficient property management experience end-to-end.
